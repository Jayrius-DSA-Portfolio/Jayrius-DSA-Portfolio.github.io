P5 IDE Robotics

This competition allowed me to improve my coding skills. There were multiple boxes on the map, each with a different colors and some with Lego inside that we had to transport to another box. Our robot used single, double, and proportional line tracing to navigate its path to the different points on the map The order of the blocks was only given on the competition day itself. During this competition, we learned how to manage the time of our robot run and learned to time each run to make sure that we could complete all the missions in the shortest amount of time. We used single, double, and p-line tracing to ensure the quickest route to each box while skipping many junctions. From this competition, I learned to always keep track of the amount of time taken to complete a run during the competition's practice time.
