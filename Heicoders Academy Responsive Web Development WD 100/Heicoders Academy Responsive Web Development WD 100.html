Heicoders Academy Responsive Web Development SE 100

In this course, I learned about HTML, CSS, and JavaScript, how we can use HTML to execute the page, CSS to add color and design, and JavaScript to give it interaction. In HTML, I learned about tags to start the webpage like !DOCTYPE html to tell the program that this is an HTML page and tags like html, head or body to address the different types of parts in an HTML program. Any code inside head are values that are not meant to be shown in the website, like adding external CSS or JavaScript code into the website or adding internal CSS and JavaScript. I also learned tags like h1 ... h6 that can create different sizes for headings or sub-headings. tags like p to create paragraphs, /br to separate words in a paragraph, a href="" for links, img scr="" alt="" for images, ol or ul for unordered and ordered lists and li to add items to the lists, input... for textboxes, checkboxes and many more. I learned about the three different kinds of listing, arrays, objects, or JSON. In CSS, I learned about classes, IDs, and names that can be accessed in HTML code to give design. I learned how to create boxes to store words inside them and how to use the margin to move them around the webpage. In JavaScript, I learned how to make buttons interactive with functions and how to create variables using let const, or var. I learned about how I can use DOM to show those variables on the website. I also learned about application programming interfaces, also known as APIs, that give you live data like the weather, temperature, or stocks.
