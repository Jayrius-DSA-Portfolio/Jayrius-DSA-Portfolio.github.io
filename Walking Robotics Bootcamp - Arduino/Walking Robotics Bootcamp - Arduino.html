Walking Robotics Bootcamp

During this two-day boot camp, I learned about six-legged robots. I learned how to make a robot walk using 3 motors on each leg in a pattern that lifts, shifts, and drops down using 3 legs first then the other three legs. I also thought of how to solve some real-world problems using walking robots and explore dangerous areas that are not safe for humans to go to. The robot was coded with Arduino in C++. It could move on every terrain that was available to me around the streets. This boot camp enhanced my Arduino knowledge and coding algorithms which can benefit me in all my competitions.
